### A quick way to chat

This is a quick chat app for ALX and Holberton Software engineers. No hassles.

## Installation

OS X & Linux:

```sh
apt-get install fast-chat /* In development */
```

Windows:

```sh
win-get install fast-chat /* In development */
```
